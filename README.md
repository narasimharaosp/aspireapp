# aspire finance management mobile interface

## Build Setup

```bash
$ node -v
v14.17.0

$ yarn --version
1.22.10

# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn dev

# unit test
$ yarn test

# build for production and launch server
$ yarn build
$ yarn start

# generate static project
$ yarn generate
```
